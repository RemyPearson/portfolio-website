This is myActivity Generator

Bored? Check out my website for ideas!

A website I created with a team of individuals from our inTech Club at Broward College. The website was created to generate activities for the user for when they are bored. We built the website from scratch and linked it to an API that generates the activities and displays them for the user. We worked on this project remotely due to COVID and it was great experience for collaborating with others in a remote setting.

Languages used are HTML, SCSS and JavaScript.
